Deploy Static Website on AWS

Overview:
This is a demo project to deploy a static website on AWS using services including S3, IAM and Cloudfront.
The website can be accessed using the Cloud front domain name URL, the S3 bucket website endpoint or the "index.html" bucket object URL.

URLs:
Cloudfront Domain Name URL: https://d2xlxz7dedlwuc.cloudfront.net/
S3 Website endpoint URL: http://udacitywebsitedeploymentproject.s3-website-us-east-1.amazonaws.com/
S3 bucket object URL: https://udacitywebsitedeploymentproject.s3.amazonaws.com/index.html
